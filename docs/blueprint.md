# **App Name**: Apex Configurator

## Core Features:

- Service Catalog: Display a catalog of available services.
- Color Customization: Allow users to customize the car's color using a color picker.
- Parts Selection: Enable users to select and preview different car parts (e.g., wheels, spoilers).
- Live Preview: Showcase a live preview of the customized car in real-time.
- AI-Powered Modification Suggestions: Generate tool that suggests relevant modifications to a particular configuration using past customer preferences.
- Login & Registration (Dummy): Implement login and registration forms for users to save their builds. Note: Due to project constrains, data won't persist.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of trust and technology.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Vibrant orange (#FF5722) to highlight key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern aesthetic suitable for all text.
- Use clear, outlined icons to represent different modification options and services.
- Maintain a clean and spacious layout to avoid overwhelming the user with customization options.
- Use subtle animations to transition between car views and display modification previews.
